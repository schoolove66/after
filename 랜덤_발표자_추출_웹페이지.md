# 랜덤 발표자 추출 웹페이지

## 개요

학생들의 발표순서를 랜덤 추출

---

## 목표 및 대상

초등학생들 발표순서를 랜덤 뽑기함

---

## 핵심 기능

- 발표자를 랜덤으로 뽑아서 보여주는 기능
  - 추출 인원을 선택할 수 있는 기능
- 학생 명단을 추가하는 기능
  - 한명 혹은 여러명을 일괄로 추가할 수 있는 기능
- 교사가 마음대로 몰래 발표자 순서를 정하는 기능

---

## 화면 구성

월드컵 기념 축구선수들이 공을 차는데 이 학생의 이름이 적힌 공이 골대를 들어가서 골키퍼가 공을 들면 뽑힌것

---

## 기술 스택

- 프론트엔드
  - react + vite
